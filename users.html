Related projects

Software projects using Dask

  • Pangeo: A community effort for big data geoscience in the cloud.
  • XArray: Brings the labeled data power of pandas to the physical sciences, by providing N-dimensional variants of the core pandas data structures.
  • Intake: A lightweight package for finding, investigating, loading and disseminating data.
  • Scikit-image: A collection of algorithms for image processing in Python.
  • Datashader, geoviews, holoviews: Visualization packages for large data.
  • SatPy: Library for reading and manipulating meteorological remote sensing data and writing it to various image and data file formats.
  • Prefect Core: A workflow management system, designed for modern infrastructure.
  • Streamz: A package to help build pipelines to manage continuous streams of data.
  • Scikit-allel: Provides utilities for exploratory analysis of large scale genetic variation data.
  • Featuretools: A Python framework for automated feature engineering.
  • Stumpy: A Python library that can be used for a variety of time series data mining tasks.
  • tsfresh: Automatic extraction of relevant features from time series.
